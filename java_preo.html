🗺 How to crack this — starting from zero Java/React background

JD fit check: INFSYS-EXTERNAL-247862 lists just Core Java + React JS as primary skills — pretty generic Infosys template. Your resume already covers that and more (Spring Security, JWT, RBAC, WebSockets, FCM) — lead with those as differentiators. One thing to prep an answer for: the JD asks 2-3 YOE, your resume totals ~1.5 years — frame it around ownership/depth of what you built, not tenure length.
Week 1 — Core Java & OOP
  • OOP pillars, this/super, static vs instance
  • String pool, StringBuilder, immutability
  • Exception handling, custom exceptions
  • Collections: List/Set/Map internals, HashMap working
  • Java 8: Streams, Lambdas, Optional, functional interfaces
Week 2 — Spring Boot / REST / JPA
  • Spring core: IoC, DI, bean scopes
  • Spring Boot annotations, @RestController, @Service
  • Building REST APIs (GET/POST/PUT/DELETE), status codes
  • Spring Data JPA / Hibernate mappings, @OneToMany etc.
  • Exception handling in REST (@ControllerAdvice), validation
Week 3 — React JS
  • JSX, components, props vs state
  • Hooks: useState, useEffect, useContext, useMemo
  • Lifecycle (class) vs hooks (functional) mapping
  • Calling REST APIs with fetch/axios, loading/error states
  • Basic Redux or Context API state management
Week 4 — Integration & Mock Rounds
  • SQL joins, indexing, normalization basics
  • Microservices basics: what/why, REST vs messaging
  • Rebuild 1–2 small full stack CRUD apps end-to-end
  • Prepare 2–3 STAR-format project stories from your Zoho work
  • Two full revision passes through this tracker (use Rev1/Rev2)
